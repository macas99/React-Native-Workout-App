# React Native Workout Tracker App

React Native app for tracking workouts.

#### Pre-requisites

- Node.js
- npm
- Expo CLI

#### Run in Expo Go app:

1. **Install Dependencies**

    ```bash
    npm install
    ```
   
2. **Start the App**

    ```bash
    npx expo start
    ```
